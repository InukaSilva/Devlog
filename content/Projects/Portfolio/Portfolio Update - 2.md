I got my minimum viable version of my portfolio up for September, and while I did like the look of it, there was just something that felt like it was missing..... so I decided to re-remake my portfolio 😔

My main issue with what I had been working on was that it felt empty with out a model present. This didn't help as I wasn't sure exactly what I wanted to fill that space. Originally I wanted to put a sampling but I hadn't put enough time into Blender and got stuck when trying to get the material and shading loaded with React Three Fiber. 

I then tried to use BrickLink's Studio 2.0 to make a Lego Spaceship model, and while I did make a model but wasn't sure how to load it in once I've made it. While I think I will eventually add a model, I think it is best for me to work with something that can stand by itself without needing one to support it.

![[rocket_model.png]]

One other nitpick about my previous design was the lack of expandability. Since I was doing things minimalistic, it meant if I wanted to add things, it would be very hard to.

So in the end I started with a blank slate, and just stuck to using text, tailwind, and making things as simple as I can. What I arrived was something along the lines of this:

![[home page for portfolio 2.0.png]]

I will continue to add features and other things to make it less flat, but for now I think it does what it needs to well.
